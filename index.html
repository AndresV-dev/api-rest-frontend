Update User

https://dominio.com.com/v1/auth/update/${uuid_user}

Description

Endpoint utilizado para actualizar la informacion alojada en base de datos de un usuario, se requiere Authorization Token para esta operacion asi como el identificador Unico del usuario (UUID)

body

Solo se deberan mandar los campos que se quieren actualizar asi como uuid en la url (por lo menos 1 campo)

{
  "nombre" :"Justin",             // opcional
  "apellido_paterno" :"Martinez", // opcional
  "apellido_materno" :"Roa",      // opcional
  "rol" : "USER",                 // opcional
  "email" :"Justin",              // opcional
}

Request Example

Request Body

    {       
      "name": "Justin", 
      "apellido_paterno": "Roa"
    }
×

Response Example

{
  "id" : 1052,
  "uuid" : "7a55fac4-9e34-4f9a-a95b-c92bdb962ac5",
  "nombre" : "Justin",
  "apellido_paterno" : "Roa",
  "apellido_materno" : "Smith",
  "email" : "johnmilton@gmail.com",
  "rol" : "OPERACIONES",
  "enabled" : true,
  "username" : "johnmilton@gmail.com",
}
